# 成长姓名牌生成器 - 重构设计文档

## 项目概述

### 项目重新定位
- **原名称**：幼儿手绘姓名牌生成器
- **新名称**：成长姓名牌生成器
- **定位**：陪伴学习生涯的个性化姓名牌生成工具

### 目标用户
1. **家长**：为不同年龄段的孩子制作姓名牌
2. **教师**：批量制作班级学生姓名牌
3. **学生**：高中阶段可自主使用，自我表达

### 核心体验目标
- **视觉惊艳**：暖橙系配色、活泼童趣风格、圆角渐变、装饰元素
- **智能省心**：根据年龄自动推荐合适的风格，减少选择负担
- **自由可控**：不满意推荐时可手动调整所有参数

## 设计风格

### 整体风格方向
**活泼童趣**：明亮温暖，强调可爱元素。圆角、渐变、卡通装饰，动画反馈明显。适合全年龄段，孩子也会喜欢。

### 界面布局结构
**左侧固定面板**：控制面板固定在左侧，预览区域占据主要空间。

### 控制面板组织
**分组卡片式**：每个功能区块独立卡片，有彩色标题区分。视觉清晰，区块之间有明显分隔。

### 配色系统（暖橙系）

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主色 | `#ff9800` | 按钮、标题强调、选中状态 |
| 主色浅 | `#ffcc80` | 卡片背景、hover 状态 |
| 辅色 | `#ffe0b2` | 装饰区域、次要背景 |
| 背景 | `#fff8f0` | 页面整体背景，温暖米白 |
| 文字主 | `#4a3f35` | 深棕，比纯黑温暖 |
| 文字辅 | `#8b7355` | 中棕，次要文字 |
| 卡片白 | `#ffffff` | 功能区卡片背景 |

## 功能模块设计

### 面板结构（分组卡片式）

每个区块为独立卡片，带彩色小标签区分：

1. **年龄段选择（橙红标签）**
   - 四个年龄段按钮：幼儿园 / 小学 / 中学 / 高中
   - 点击后自动更新推荐主题和装饰密度
   - 选中状态用主色填充，带图标装饰

2. **主题选择（金黄标签）**
   - 8个主题以图标网格展示（4×2 或 2×4）
   - 当前推荐主题默认选中，其他可手动切换
   - 切换后实时预览更新

3. **内容输入（天蓝标签）**
   - 姓名输入框（必填，最长10字）
   - 班级/年级输入框（可选，最长15字）
   - 寄语/座右铭输入框（新增，可选，最长20字，适合高中用户）
   - 照片预留区开关（幼儿园/小学阶段显性提示）

4. **样式调整（浅绿标签）**
   - 姓名字体大小滑块（200-400px）
   - 姓名位置滑块（上下位置，30%-70%）
   - 装饰密度切换（繁复/标准/简约/极简，四档）

5. **导出操作（主色按钮区）**
   - 导出 PNG 按钮（单张高清）
   - 导出 PDF 按钮（A4 横向，支持批量）
   - 批量生成入口（弹窗输入多个姓名列表）

## 风格系统

### 年龄段自动推荐逻辑

| 年龄段 | 默认装饰密度 | 默认主题 | 设计理念 |
|--------|--------------|----------|----------|
| 幼儿园 | 繁复 | 卡通动物 | 色彩丰富，装饰多，吸引注意力 |
| 小学 | 标准 | 自然植物 | 适度装饰，开始有秩序感 |
| 中学 | 简约 | 几何科技 | 干净利落，有个性表达 |
| 高中 | 极简 | 文艺水墨 | 克制成熟，接近成人审美 |

### 装饰密度定义

- **繁复**：边缘多装饰图案（8-12个），背景有纹理，边框有阴影层次
- **标准**：适中装饰（4-6个），背景简洁渐变，边框单层
- **简约**：少量装饰（2-3个），背景纯净，边框细线
- **极简**：无装饰或1个点缀，背景纯色，无边框或极细线

### 共享主题池（8个）

所有年龄段共享同一主题池，用户自由选择不受年龄限制：

1. **卡通动物**：小熊、兔子、猫咪等可爱动物轮廓
2. **自然植物**：叶子、花朵、藤蔓等自然元素
3. **星空宇宙**：星星、月亮、行星、火箭
4. **海洋世界**：海浪、贝壳、小鱼、珊瑚
5. **音乐艺术**：音符、画笔、乐器轮廓
6. **几何科技**：圆形、三角、线条组合
7. **文艺水墨**：淡墨晕染、书法笔触
8. **极简线条**：简单几何线条点缀

### 智能推荐机制

- 根据年龄自动推荐合适的主题和装饰密度
- 用户可手动覆盖推荐，切换任意主题或密度
- 切换年龄时重置为该年龄段的推荐值

## 交互与动效

### 微交互规范

| 交互场景 | 动效 | 说明 |
|----------|------|------|
| 年龄段切换 | 面板滑入 + 主题刷新 | 预览区短暂淡出后重新渲染 |
| 主题切换 | 装饰元素飘入动画 | 新装饰从边缘淡入滑至位置 |
| 滑块调节 | 实时预览 + 滑块轨迹色变 | 调整时 Canvas 即时更新 |
| 生成按钮点击 | 弹跳 + 粒子散开 | 成功生成后短暂庆祝动效 |
| 导出按钮点击 | 进度条 + 完成徽章 | 导出过程显示进度动画 |
| 卡片 hover | 轻微上浮 + 阴影加深 | 提升层次感 |
| 输入框 focus | 边框变橙 + 柔光晕 | 温暖反馈感 |

### 动画时长标准

- 微交互：150-250ms（快速响应感）
- 过场动画：300-500ms（流畅但不拖沓）
- 庆祝动效：800-1200ms（足够被感知）

### Easing 函数

- 入场/弹出：`cubic-bezier(0.34, 1.56, 0.64, 1)`（轻微过冲）
- 位移/滑动：`cubic-bezier(0.4, 0, 0.2, 1)`（Material 标准）
- 淡入淡出：`ease-out`

### 动效克制规则

- 高中年龄段模式下，动效减少 50%，更加克制
- 用户连续操作时（如滑块拖拽），禁用装饰飘入动画
- 导出过程中冻结预览，避免渲染冲突

### 核心模块职责

| 模块 | 职责 | 依赖 |
|------|------|------|
| state.js | 管理全局状态，提供 getState/setState | 无 |
| canvas.js | Canvas 初始化、尺寸计算、清空 | state.js |
| renderer.js | 组装绘制流程，调用风格模块 | canvas.js, state.js, styles/ |
| styles/index.js | 风格注册表，根据年龄段选择风格 | styles/*.js |
| themes/*.js | 装饰元素位置、类型数据 | 无 |
| ui/*.js | UI 组件交互、更新状态 | state.js |

### 状态对象结构

```javascript
const state = {
  ageGroup: 'kindergarten',  // 年龄段: kindergarten | primary | middle | high
  theme: 'animals',          // 主题
  density: 'rich',           // 装饰密度: rich | standard | simple | minimal
  name: '',                  // 姓名（最长10字）
  className: '',             // 班级（最长15字）
  motto: '',                 // 寄语（最长20字）
  showPhoto: false,          // 照片预留区开关
  fontSize: 250,             // 姓名字体大小（200-400）
  namePosition: 50,          // 姓名位置百分比（30-70）
  layout: 1                  // 每页数量：1 | 2 | 4
};
```

## 设计决策记录

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 整体风格 | 活泼童趣 | 适合全年龄段，界面温暖明亮 |
| 布局结构 | 左侧固定面板 | 保持熟悉感，面板紧凑有装饰感 |
| 面板组织 | 分组卡片式 | 功能清晰区分，视觉层次分明 |
| 配色 | 暖橙系 | 温暖阳光，男女通用，活力积极 |
| 风格维度 | 年龄段 + 主题 + 装饰密度 | 灵活组合，满足不同需求 |
| 主题池 | 共享（不按年龄区分） | 用户自由选择，不受年龄限制 |
| 推荐机制 | 智能推荐 + 手动调整 | 简化入门，保留自由度 |
| 年龄段划分 | 幼儿园/小学/中学/高中 | 陪伴完整学习生涯 |
