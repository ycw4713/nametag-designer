# 幼儿手绘姓名牌生成器设计文档

## 背景

幼儿老师需要在六一儿童节时，为每个孩子准备手绘风格的姓名牌（A4纸大小）。需要一个简单易用的网页工具来生成和打印这些姓名牌。

## 目标

创建一个纯前端网页应用，让老师可以：
- 输入孩子姓名
- 选择不同的视觉风格
- 自定义姓名牌上的元素
- 导出为 PDF 或 PNG 图片便于打印

## 功能需求

### 核心功能

1. **风格选择** - 三种可选风格：
   - 可爱卡通风格：卡通动物、星星、爱心、彩虹等装饰
   - 简洁清新风格：简约线条、几何形状、干净整洁
   - 手绘艺术风格：模拟蜡笔/彩笔涂鸦效果

2. **姓名输入** - 输入孩子的姓名，显示在姓名牌中心位置

3. **元素自定义** - 用户可选择添加：
   - 班级信息（如"大一班"、"小二班"）
   - 照片位置（预留区域贴照片或画头像）
   - 装饰图案（根据风格提供不同装饰选项）

4. **实时预览** - 所有修改实时反映在预览区域

5. **导出功能** - 支持两种导出格式：
   - PDF 文件（A4 尺寸，适合直接打印）
   - PNG 图片（适合分享或打印）

6. **打印布局** - 可选择每页打印数量：
   - 1个姓名牌（居中，大字体）
   - 2个姓名牌（上下排列）
   - 4个姓名牌（2×2 排列，节省纸张）

## 技术方案

### 技术栈

- **HTML + CSS + JavaScript** - 纯前端，无需构建步骤
- **Canvas API** - 绘制姓名牌内容
- **jspdf 库** - PDF 导出功能
- **Google Fonts** - 开源中文字体

### 三种风格设计详情

#### 可爱卡通风格

- **背景**：浅粉色或浅蓝色渐变
- **装饰元素**：
  - 卡通动物图案（小熊、兔子、小猫）
  - 星星、爱心、彩虹
  - 云朵、气球
- **边框**：圆润边框，带阴影效果
- **颜色**：粉红、粉蓝、粉黄为主

#### 简洁清新风格

- **背景**：纯白色或淡灰色
- **装饰元素**：
  - 简约线条边框
  - 小图标（树叶、花朵轮廓）
  - 几何形状装饰
- **边框**：细线条边框，干净整洁
- **颜色**：白、灰、淡蓝为主

#### 手绘艺术风格

- **背景**：纸张纹理效果（米黄色）
- **装饰元素**：
  - 模拟蜡笔/彩笔绘制的涂鸦
  - 手绘花朵、太阳
  - 不规则线条装饰
- **边框**：手绘不规则边框
- **颜色**：蜡笔色调（红、蓝、绿、黄）

### A4 纸尺寸

- 宽度：210mm
- 高度：297mm
- Canvas 像素比例：按 96 DPI 计算，约 794 × 1123 像素

### 导出实现

**PNG 导出：**
```javascript
// 使用 Canvas toDataURL
const dataUrl = canvas.toDataURL('image/png');
// 创建下载链接
```

**PDF 导出：**
```javascript
// 使用 jspdf 库
const pdf = new jsPDF('p', 'mm', 'a4');
// 将 Canvas 图像添加到 PDF
pdf.addImage(dataUrl, 'PNG', 0, 0, 210, 297);
pdf.save('nametag.pdf');
```
